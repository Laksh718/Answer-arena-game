## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8080
   ```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features in Detail

- **User Profiles**: Create your quiz profile with a custom avatar
- **Interactive Questions**: Multiple choice questions with immediate feedback
- **Progress Tracking**: Visual progress bar to track your quiz journey
- **Performance Stats**: Get detailed results including accuracy and grade level
- **Responsive Design**: Works seamlessly on both desktop and mobile devices
